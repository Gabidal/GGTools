# GGTools
Packaging manager for GGUI, GGDirect, Evie and more.

Run `./build.sh` to fetch latest code changes from main branches on GGUI and GGDirect and generate deb and rpm packages. 

If you need the public key to run this locally, you can fetch it from: https://github.com/Gabidal.gpg, Although preferably use this project if you are an actual contributor to the GG ecosystem, then you can just use your own authorized keyring's.
